    Seleku with dezui

    membuat project pertama dengan seleku dan dezui, seleku merupakan framework buatan orang indonesia maupun dezui sehingga kami telah bekerja sama dengan developer dezui dalam tutorial ini

    apa itu dezui?

    DezUI is a lightweight, minimalist & elegant UI to start a project with a minimalist and elegant design. Checkout
    https://dezui.netlify.app for documentation and details

    seleku sendiri cukup mudah untuk melakukan manipulasi terhadap output nya melalui precompiler API nya ( seleku API ) sehingga sangat di mungkinkan framework sejenis dezui untuk dapat bekerja di atas seleku

    untuk memulai project menggunakan dezui dan seleku anda cukup membuat template project dengan mengetikan
    seleku di command line anda

    silahkan pilih seleku with HMR template kemudian anda silahkan lakukan inisaliasi project

    kemudian silahkan instalasi dezui dengan menjalankan

    npm i dezui@latest

    kemudian silahkan buka file app.seleku yang terdapat di dalam folder src kemudian silahkan lakukan import seperti berikut

    <script>
	// dezui menggunakan file .css sehingga
	// anda dapat melaukan import seperti di 
	// bawah ini
	import * as __dezui from "dezui/core.css"	

</script>

    kemudian anda dapat menggunakan class yang terdapat di dezui di dalam file app.seleku sebagai contoh

    app.seleku

     <div>
  <nav>
	 <section class="navbar-section">
		  <a class="navbar-logo" href="/">Brand</a>
		  <a href="#">Link 1</a>
		  <a href="#">Link 2</a>
		  <a href="#">Link 3</a>
	 </section>
 </nav>
</div>
 
<script>
	// dezui menggunakan file .css sehingga
	// anda dapat melakukan import seperti di 
	// bawah ini
	import * as __dezui from "dezui/core.css"	

</script>

    silahkan save file app.seleku kemudian anda akan melihat hasil di web anda sekarang telah terdapat navbar yang di style menggunakan dezui framework
